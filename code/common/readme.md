### common 层

common 层的工作是设计通用、可复用的基础组件和工具。

#### 曲线 Curve 类

生成平滑的曲线函数。

目前使用 $y=-x+\sin x$ 作为模拟，向右下方延伸的斜坡。

提供 $y(x)$ 和 $y'(x)$ 两个接口。

#### 物理实体 PhysicsEntity 类

表示具有物理性质（位置、速度、加速度）的实体，但不考虑质量、力的属性。

角色、企鹅等都属于这类实体。

除了物理性质的获取和设定以外，还提供在短时间内改变运动状态 `update(deltaTime)` 的接口。

#### 计时器 Timer 类

提供以秒为单位的计时。

用于帧率的控制。

#### 向量 Vector 类

支持二维向量的基础操作（加、减、乘除标量、内积、外积、模长、标准化等）。
